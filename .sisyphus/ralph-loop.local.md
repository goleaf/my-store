## Subscriptions — List Page Page heading reads Subscriptions. On the right is a button labeled New Subscription. Filter bar contains: an Organization search field, a Plan dropdown, a Status dropdown, and a special filter labeled Expiring Within with options 7 Days, 14 Days, 30 Days, 60 Days, Any Time. Table columns are: Organization, Plan, Status, Start Date, Expiry Date, Properties Used, Tenants Used, Date Created. Properties Used shows the fraction used versus the plan limit such as 7 of 10 and turns red if the organization is at the limit. Tenants Used shows the same pattern. Upgrade Plan opens a form overlay showing the current plan and a dropdown to select the new plan, with a note showing the new limits. Suspend and Cancel both ask for confirmation with a description of consequences.
